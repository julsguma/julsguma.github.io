Real-Time Pencil Rendering

About

Team: Individual project

Tecnhology: Custom C++ engine made from scratch

High concept : Final project for my CS562: Advanced Real-Time Rendering Techniques class. The project consists on rendering 3D objects in pencil style drawing. This tries to simulate the human way of drawing things.


Steps

Contour detection

First of all I detect the contours using Sobel filter and normals.







Contour distortion

I distort the contours using a distortion plane. For that, I divide a plane in uniform rectangles, adding small errors to the texture coordinates of those rectangles. In this way, when I display the contour detection image onto that plane, the errors in the UVs will add the distortion effect.





Multiple contours

In order to simulate overlapping strokes, I add multiple distorted contours.






Texture generation

I create a 3D texture out of a single texture repeated multiple times, each one of them darker than the previous one. Depending on the light at each pixel, I select a specific layer of the 3D texture.
